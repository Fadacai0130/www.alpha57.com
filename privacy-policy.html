1. Introduction

Stock Market Today ("we", "our", "the extension") is a Chrome extension designed to provide real-time stock market information to users. We are committed to protecting your privacy and being transparent about our data practices.

Privacy-First Approach

We DO NOT collect, store, transmit, or sell any personal information.

2. Data Collection

2.1 What We Access

  • Public Financial Market Data: The extension fetches publicly available stock market data from Yahoo Finance API
  • Local Storage: Cached market data is stored locally on your device for faster loading and offline access

2.2 What We DO NOT Collect

  • Personal identification information (name, email, phone number)
  • Browsing history or website visits
  • Cookies from other websites
  • Login credentials or passwords
  • Payment or financial account information
  • Geographic location or IP address
  • Camera, microphone, or device sensors
  • Contacts, calendar, or personal files
  • User behavior analytics or tracking data

3. Permissions Explained

Chrome requires us to declare permissions. Here's exactly why we need each one and how we use them:

Storage Permission

What it allows: Saving data locally on your device using Chrome's Storage API

Why we need it: To cache stock market data temporarily for improved performance and offline access

What we store:

  • Stock symbols (e.g., AAPL, MSFT, ^GSPC)
  • Current stock prices
  • Price changes and percentages
  • Timestamp of last data fetch

Access: Only this extension can access this data. It stays on your device and is never transmitted to us or any third party.

Host Permission: query1.finance.yahoo.com

What it allows: Making API requests to Yahoo Finance servers

Why we need it: To fetch real-time stock market data from Yahoo Finance's public API

Data sent: Only stock ticker symbols (e.g., AAPL, GOOGL, ^GSPC)

Data received: Public market data including prices, changes, and trading volumes

Frequency: Every 5 minutes when the extension is active, or when you manually refresh

4. Third-Party Services

4.1 Yahoo Finance API

We use Yahoo Finance's public API to retrieve stock market data. This is publicly available information that anyone can access.

  • Service Provider: Yahoo (Verizon Media)
  • Data Accessed: Public stock prices and market indices
  • Personal Data Shared: None
  • Privacy Policy: Yahoo Privacy Policy

4.2 Alpha57.com

The extension includes a link to Alpha57.com for additional market analysis. This is a standard hyperlink that opens in a new browser tab.

  • Automatic Data Sharing: None - data is only shared if you click the link
  • What Happens: Clicking opens Alpha57.com in a new tab like any website link
  • Privacy Policy: Alpha57.com's privacy policy applies when you visit their website

4.3 No Analytics or Tracking

We do NOT use:

  • Google Analytics or any analytics services
  • Third-party advertising networks
  • Tracking pixels or beacons
  • Social media plugins that track users
  • Error reporting services that collect personal data

6. Data Security

While we don't collect personal data, we still implement security best practices:

6.1 Technical Measures

  • Local Storage Only: All cached data stays on your device
  • No Remote Servers: We don't operate servers that store user data
  • HTTPS Only: All API calls use secure HTTPS connections
  • Chrome Sandboxing: Extension runs in Chrome's security sandbox
  • Content Security Policy: Prevents unauthorized code execution

6.2 Code Transparency

  • No code obfuscation - all JavaScript is readable
  • Open for review by Chrome Web Store security team
  • Minimal external dependencies
  • Regular security updates

7. User Control

You have complete control over the extension and your data:

7.1 Managing the Extension

  • Disable: Go to chrome://extensions/ and toggle the extension off
  • Remove: Click "Remove" to uninstall completely
  • Review Permissions: View all permissions on the extensions page

7.2 Data Management

  • Clear Cache: Uninstalling the extension removes all cached data
  • View Storage: Use Chrome DevTools to inspect local storage
  • Prevent Storage: Disable the extension to stop data caching

7.3 Your Rights

Under applicable data protection laws, you have the right to:

  • Know what data is collected (disclosed in this policy)

8. Children's Privacy

This extension is designed for adults interested in financial markets. We do not knowingly collect information from children under 13 years of age.
